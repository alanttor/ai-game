# AI Game & Tech Collections

A collection of AI-assisted interactive projects, full-stack 3D games, text adventures, and utility services.

This repository serves as a centralized hub for exploring various programming paradigms (from pure static HTML/Canvas simulations to Spring Boot + Three.js full-stack 3D games).

---

## 🎮 Project Directory

| Project Name | Technology Stack | Description | Directory Link |
| :--- | :--- | :--- | :--- |
| **HTML 互动项目展示** | HTML5, Canvas, LocalStorage | 50+ pure-frontend physics, biology, and math simulations with theme toggling. | [prompt-word-demo](./prompt-word-demo) |
| **纸人归魂 (Chinese Horror)** | Python Flask, Web Audio API | Session-based text adventure game with dynamically synthesized horror audio. | [chinese_horror_game](./chinese_horror_game) |
| **僵尸世界大战 (Zombie World War)** | Vue 3, Three.js, Spring Boot 3 | Full-stack 3D first-person shooter game featuring JWT authentication and leaderboard. | [zombie_world_war](./zombie_world_war) |
| **凡人修仙 - 进击的巨人** | Python, Ursina 3D Engine | A 3D action game prototype containing ODM physics and Titan behavior logic. | [Fanren_Attack_on_Titan](./Fanren_Attack_on_Titan) |
| **智能链接摘要** | Python Flask, BS4 | Web scraping service to parse Bilibili, Zhihu, and CSDN articles to summaries. | [Summary_of_Link_Content](./demo/Summary_of_Link_Content) |

## 🚀 Central Entry Portal

You can launch a unified visual directory index by hosting the root [index.html](./index.html) file on a static server.

### Quick Start with Node.js
If you have Node.js installed, run:
```bash
npx serve .
```
Open [http://localhost:3000](http://localhost:3000) to browse and select projects.

---

## 🛠️ General Prerequisites
To run the full suite of backend-enabled projects, you will need:
- **Python 3.8+** (for Flask & Ursina projects)
- **Node.js 18+** & **npm** (for Vue 3/Three.js frontend development)
- **Java JDK 17** & **Maven 3.8+** (for Spring Boot backend service)
- **MySQL 8.0+** (for game database storage)
