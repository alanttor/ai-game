# 智能链接摘要 Web 应用

支持 B站视频（含CC字幕）、知乎、CSDN等文章链接的智能摘要生成工具。

## 功能特性

- ✅ **B站视频摘要**：支持基于视频标题、简介、CC字幕生成精炼摘要
- ✅ **文章链接摘要**：支持知乎、CSDN等平台文章的完整内容提取与摘要
- ✅ **智能识别**：自动识别链接类型，选择最佳提取策略
- ✅ **简洁精炼**：生成3-5句核心摘要 + 关键要点提取
- ✅ **本地运行**：无需服务器，本地即可部署使用

## 技术限制说明

### B站视频内容提取
- **有CC字幕的视频**：可完整获取字幕内容，摘要精度高
- **无CC字幕的视频**：基于标题、简介、弹幕碎片生成摘要，精度有限
- **原因**：B站不对外开放完整字幕API，CC字幕是官方提供的可访问字幕资源

### 文章链接提取
- **知乎/CSDN**：可提取完整正文内容
- **其他网页**：基于通用网页解析，部分网站可能有反爬虫限制

## 安装与运行

### 环境要求
- Python 3.7+
- pip

### 安装步骤

1. **克隆或下载项目文件**

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **启动应用**
```bash
python app.py
```

4. **访问应用**
打开浏览器访问：`http://localhost:5000`

## 使用方法

1. 在输入框中粘贴链接（支持 B站视频、知乎、CSDN等）
2. 点击"生成摘要"按钮
3. 等待系统分析内容（通常5-15秒）
4. 查看生成的摘要和关键要点
5. 可点击"复制摘要"按钮导出结果

## API 接口

### POST /api/summarize

生成链接摘要

**请求参数：**
```json
{
  "url": "https://www.bilibili.com/video/BV1xxx"
}
```

**响应示例：**
```json
{
  "success": true,
  "data": {
    "source_type": "bilibili",
    "title": "视频标题",
    "url": "https://www.bilibili.com/video/BV1xxx",
    "summary": "内容摘要...",
    "key_points": ["要点1", "要点2", "要点3"],
    "has_subtitle": true,
    "content_length": 2500,
    "platform": "B站视频",
    "time": "2026-01-23 10:30:00"
  }
}
```

### GET /api/health

健康检查接口

**响应示例：**
```json
{
  "status": "ok",
  "timestamp": "2026-01-23 10:30:00"
}
```

## 摘要生成说明

当前版本使用**基于规则的摘要算法**，特点：
- 快速响应，无需AI服务
- 适合快速验证和原型测试

**如需更智能的摘要**，可替换 `Summarizer` 类为以下方案：
- 接入 OpenAI GPT API
- 接入文心一言等国产大模型
- 使用本地部署的开源模型（如ChatGLM）

## 常见问题

### Q: B站视频摘要不够准确？
A: 请确认视频是否开启CC字幕。未开启字幕的视频只能基于元数据生成摘要。

### Q: 某些网页无法提取？
A: 部分网站有反爬虫机制或内容动态加载，这类页面可能无法正常提取。

### Q: 如何部署到公网？
A: 可使用云服务器（如阿里云、腾讯云）+ Nginx 反向代理，或使用内网穿透工具（如ngrok）。
